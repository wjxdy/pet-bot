## 迭代路线图

### 迭代 1: 修复基础功能（当前）
**目标**: 确保所有核心功能稳定运行
**验证标准**: 
- 输入框可以正常输入并发送
- 气泡窗口正确显示回复
- 设置窗口能正常显示和操作

### 迭代 2: UI 美化
**目标**: 改进视觉效果
**验证标准**:
- 宠物窗口圆角 + 阴影
- 气泡窗口美化（毛玻璃效果）
- 输入框无边框 + 圆角
- 动画效果流畅

### 迭代 3: 配置持久化
**目标**: 用户设置保存到本地
**验证标准**:
- 重启后设置保持不变
- 宠物位置记忆
- Agent 选择记忆
- 偏好设置生效

### 迭代 4: 功能增强
**目标**: 添加实用功能
**验证标准**:
- 消息历史记录
- 多会话支持
- 快捷指令
- 语音输入

### 迭代 5: 服务端优化
**目标**: 改进后端连接
**验证标准**:
- WebSocket 连接稳定性
- 重连机制
- 消息队列
- 错误处理
